# 📌 Project Overview

The **EventBeep Content Management & Blog Platform** is a lightweight Content Management System (CMS) designed to allow recruiters and administrators to publish career-related blogs, company announcements, and upcoming events.

Unlike the Recruiter and Student platforms, this project was built using **Next.js** and **Chakra UI** to leverage **Server-Side Rendering (SSR)**, improving page load performance, search engine visibility (SEO), and the overall reading experience.

My contribution focused on building responsive CMS interfaces, integrating backend APIs, and creating an optimized content platform for both content creators and readers.

---

# 🎯 Business Problem

The company required an integrated blogging platform where recruiters and administrators could easily publish and manage content while students could browse well-organized career articles and event announcements.

The platform needed to:

* Improve SEO through Server-Side Rendering
* Deliver faster initial page loads
* Provide a responsive reading experience
* Simplify blog and event management
* Organize content through categories and tags

# ⚡ Server-Side Rendering (SSR)

One of the primary objectives of this project was to leverage **Next.js Server-Side Rendering**.

Benefits included:

* Faster Initial Page Load
* Improved SEO
* Better Social Media Link Previews
* Enhanced User Experience
* Optimized Content Rendering
* Improved Performance for Public Pages

Using SSR ensured that blog content was rendered on the server before reaching the client, making pages more discoverable by search engines and reducing perceived loading times.
